Web Usability

Discuss web usability considered and implemented for this portfolio website

I have considered the following 10 web usability features to ensure user satisfaction:

  • Number of fonts
    • ◦ I've used times new roman throughout this website to maintain structure and professionalism.

  • Heading
    • ◦ I've made the heading(s) stand out by making its background colour different than the rest of the webpage and by making the size larger.

  • Font Style
    • ◦ I used a professional font (Times New Roman) so that the valuable information is displayed in a proper manner.

  • Size of Text
    • ◦ I've used a size for the text that is large enough to read.
    ◦ Titles and Tasks given by professor are larger than regualr text in a paragraph.
    ◦ Answers to tasks/questions are smaller to shift your focus onto the question.

  • Spacing
    • ◦ I've used etiquette spacing to my advantage by properly using break sections to ensure readability.
    ◦ Titles are properly spaced from the answer to diminish any potenital readability issues.

  • Hyper Links
    • ◦ I've used blue for hyper links as it more universally known as the hyperlink color.
    ◦ If the user hovers of the words in the navbar they will notice they are hyperlinks to other webpages on the website.

  • Colour
    • ◦ I've used a simple colour scheme that would be appropriate on a webpage.
    ◦ White for the most of the webpage.
    ◦ Grey to add emphasis onto questions and more important headings.
    ◦ Dark grey to add emphasis to the nav bar so it stands out.

  • Contrasting
    • ◦ I've made sure that the text and background do not match so the text wont get hidden into the background.
    ◦ I've made the majority of the website white for this exact reason.

  • Amount of Text
    • ◦ I've kept the amount of text as minimal as possible.
    ◦ I've also seperated the text to make it lighter to read as a user won't want to read a big paragraph.

  • Overall Consistency
    • ◦ I've ensured that the number of fonts, size of text, font style, spacing, colour, and other elements have a structure and its not lack luster.
    ◦ I've kept a consistent pattern to ensure that the user can fully understand the text and webpage
