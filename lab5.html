Reflection


I decided to go with the most simple method when it came to the design. I've never worked with p5.js before so to ensure I learn how it works and be able to apply the knowledge with formidable evidence I decided to make a simple house. In order to make the image and page more inviting, I gave the shapes that I used a colourful outlook by going with a mix of bright and dark colours. Creating the house not only refelcted that I have knowldege on p5.js but also embodied my artistic ability. There were a number of steps that I took in order to complete this task which will be explained below.

I did not have too many steps when doing this lab since it was quite a staightforward coding experience. A lot of the code was 1-2 lines max other than some small elemnts. I started off by choosing the colours that I wanted to use, so I went with the most realistic colours that I could think of by going with: beige, khaki, grey, silver, yellow, brown, and black. Then I read up on the documentation online and provided by Sunny to familiarize myself with the coding practices. First, I decided to set the size of the canvas to a square and set the colour to black to make the house pop out. Then I made the roof by using a triangle and the house by using the rectangle. After, I made the door by using another rectangle and used an ellipse to create both the moon and door knob. I made a block for the text/address on the house using a square. I then decided to create a light post by using a combination of rectangle and ellipse. I also added some basic text to see how it works. I also used 2 images which can closely reflect what the image is showing. I used an image for a moon and a shooting star. After, I ran the program to see my creation and then transferred it into the website.
