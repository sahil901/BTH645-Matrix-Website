  2. The use of audio on your website regarding usability.
    3.
    - I utilized the high-quality audio file for the best results which is why the Bollywoood song is clear.
    - I used regular frequency audio files to take into consideration people with sensitive ears.
    - I used mono when using my voice and used lossless compression since they're the most recommeneded.

  3. Any thoughts on different audio file formats, e.g., cons, pros, when to use.
    4.
    Type of File Pros Cons When to Use
    MP3 Reduced file size Potentially poor audio quality Web pages/videos
    OGG Better sound quality than MP3 even at same bit rate Conversion heavily degrades sound Royalty-free audio compression
    WAV Can easily be converted into other file type Consume big memory space and not ideal for uploading online TV, Radio, DVD

  4. Description of your editing on the file, e.g., cutting, assembly, fadein, fadeout, etc.
    5.
    - The file that was edited was the second one which was titled "Edited Bollywood Song Instrumental.
    - Seconds 1 - 3: The audio was sped up to see what difference a jump from 0 - 3 can make. The result of this change was noticeable in comparison to the old audio file.
    - Seconds 6 - 10: The tempo on the audio was sped up to help notice the difference from speed and tempo itself. Tempo felt like the base was quicker not the song itself.
    - Seconds 14 - 19: The pitch was changed to made the song much heavier on the ears and base. It was louder and much more *thump* was added to the audio file.
    - Seconds 29 - 31: The audio was faded out to make the end of the song more smooth instead of the cut which was in the original song.
